# NavzaBD Baby Shop 🛍️

A full-stack ecommerce web application for baby products.

## 🛠️ Tech Stack
- **Frontend:** Next.js, TypeScript, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT

## ✨ Features
- User authentication (register, login)
- Product listing with search & filter
- Shopping cart
- Order management
- Admin dashboard

## 🚀 How to Run Locally

### Client
cd client
npm install
npm run dev


### Server
cd server
npm install
npm run dev
